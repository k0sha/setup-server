# Changelog

## [1.0.0] — 2026-04-13

### Первый публичный релиз

**Фаза 1 — настройка сервера:**
- Timezone, hostname, создание пользователя с sudo
- Харденинг SSH (кастомный порт, отключение root и password auth)
- UFW: 80, 443, SSH, VLESS Reality, NODE_PORT, gRPC (61000)
- Опциональное отключение systemd-resolved и удаление Zabbix Agent
- Сохранение конфига после каждого введённого поля (возобновление с места остановки)
- Самоперемещение скрипта в `/opt/setup-server/` при запуске из другого места

**Фаза 2 — деплой ноды:**
- Установка Docker Engine с обработкой rate limit Docker Hub
- Деплой Remnawave Node через docker compose (NODE_PORT + SECRET_KEY)
- Watchdog-таймаут при чтении логов ноды (120 с) и Nginx (60 с)
- Установка acme.sh, DNS-проверка перед выдачей сертификата
- Деплой Nginx с самоподписанным → реальным сертификатом
- Авторенью сертификата через webroot (без остановки nginx)
- Health-check: статус контейнеров, дней до истечения SSL, порты 80/443
- deploy-info.txt с итоговой сводкой (IP, SSH, домен, порты; без паролей и ключей)

**Прочее:**
- Версионность и проверка обновлений при первом запуске из install dir
- Trap + cleanup при прерывании (Ctrl+C, ошибка)
- Поддержка частичного конфига — возобновление ввода с последнего заполненного поля
